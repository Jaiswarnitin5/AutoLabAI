# AutoLabAI 🤖

## 🚀 Overview

AutoLabAI is a full-stack AI-powered application that automates data analysis and generates intelligent insights using machine learning techniques.

## 🧠 Features

* AI-based data processing
* Automated report generation
* Interactive frontend dashboard
* Backend API for ML operations

## 🛠 Tech Stack

* Frontend: React.js
* Backend: Python (Flask)
* Database: MongoDB
* Machine Learning: Pandas, NumPy, Scikit-learn

## 📂 Project Structure

* `frontend/` → User Interface
* `Backend/` → API & ML logic
* `requirements.txt` → Dependencies

## ▶️ How to Run

### Backend:

pip install -r requirements.txt
python app.py

### Frontend:

cd frontend
npm install
npm start

## 🔐 Environment Variables

Create a `.env` file:

MONGO_URI=your_mongodb_uri
GROQ_API_KEY=your_api_key
