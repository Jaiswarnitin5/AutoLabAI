# AutoLabAI 🤖

## 🚀 Overview

AutoLabAI is an AI-powered application that generates lab practicals, reports, and supports viva preparation. It helps students quickly create structured lab content and practice viva questions using AI.

## 🧠 Features

* 📄 Generate lab practicals using AI
* 📝 Automated lab report creation
* 🎤 Viva practice with AI-generated questions
* 💡 Intelligent responses based on user input
* 🌐 User-friendly frontend interface

## 🛠 Tech Stack

* Frontend: React.js
* Backend: Python (Flask)
* Database: MongoDB
* AI Integration: Groq API

## 📂 Project Structure

* `frontend/` → User Interface
* `Backend/` → API & AI logic
* `requirements.txt` → Dependencies

## ▶️ How to Run

### Backend:

pip install -r requirements.txt
python app.py

### Frontend:

cd frontend
npm install
npm start

## 🔐 Environment Variables

Create a `.env` file:

MONGO_URI=your_mongodb_uri
GROQ_API_KEY=your_api_key

## 🎯 Use Case

* Students can generate lab practicals instantly
* Practice viva questions before exams
* Save time in report writing
